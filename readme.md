# jQuery DataTables

## 開發環境
* Node.js: 14.17.5
* electron: 13.2.2
* electron-builder: 22.11.7
* jQuery: 3.5.1
* jQuery dataTables: 1.11.0
* jQuery dataTables scroller: 2.0.5

## 說明
* /build: 存放打包後的安裝檔(包含免安裝檔),輸入指令'node build.js'就會開始打包)
* /icons: 存放icon圖片
* /node_modules: 存放套件(請自行安裝electron)
